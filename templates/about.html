About PacketGuard

Network intrusion detection mini-project

What This Is

PacketGuard classifies network traffic into Normal, DoS, Probe, R2L, and U2R categories using machine learning trained on the NSL-KDD dataset — a standard academic benchmark for intrusion detection research.

Rather than just training a model and reporting an accuracy number, this project builds a full interactive tool: live classification, model comparison, explainability, and an unsupervised anomaly detection path.

Key Finding

R2L and U2R attack recall stays low across every model tested (2-12%), even after SMOTE oversampling and hyperparameter tuning. Feature importance analysis showed models lean on traffic-volume features (byte counts, connection counts) rather than login/content features that R2L attacks actually depend on — indicating a data-level generalization ceiling rather than a tuning problem, since KDDTest+ intentionally includes attack subtypes absent from training data.

Tech Stack

Python, scikit-learn, Flask, Chart.js. Trained on NSL-KDD's KDDTrain+/KDDTest+ split. Deployed on Render.
