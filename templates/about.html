About PacketGuard

Network intrusion detection mini-project

What This Is

  • Classifies network traffic into Normal, DoS, Probe, R2L, U2R
  • Trained on NSL-KDD, the standard academic IDS benchmark
  • Full interactive tool — not just a notebook accuracy score
  • Live classification, model comparison, explainability, and anomaly detection in one dashboard

Model Comparison Results

Random Forest (baseline)74.41%
RF + SMOTE + GridSearch74.01%
Decision Tree + SMOTE ★80%
DT + SMOTE + GridSearch74.43%
RF + DT Voting Ensemble76.27%

Key Finding

  • R2L/U2R recall stays low (2–12%) across every model, even after SMOTE and tuning
  • Feature importance shows models lean on traffic-volume features, not login/content features R2L needs
  • Points to a data-level generalization ceiling, not a tuning problem
  • KDDTest+ intentionally includes attack subtypes absent from training data
