What It Does

Multi-Model Comparison

Five models tried side by side on the same NSL-KDD split.

Multi-Model Comparison

Random Forest, SMOTE-balanced variants, a Decision Tree, and a voting ensemble were all trained on the same data and evaluated on the same held-out test set — not just the winner, so you can see what tuning actually changed and what it didn't.

Explainability

Every prediction comes with a reason, not just a label.

Explainability

Feature-importance scores show which traffic attributes the model actually weighed — usually volume and connection-count features — which is also how we found where the model was falling short.

Anomaly Detection

A second, independent pass on the same traffic.

Anomaly Detection

An Isolation Forest runs alongside the classifier and flags statistically unusual flows without needing a label for them — it won't tell you what kind of attack it is, but it can catch things the classifier never saw in training.

Live Classification
